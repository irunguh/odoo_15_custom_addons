Sales Order Loyalty Rewards and Redeem Management Odoo Apps

Sales Loyalty Rewards and Redeem Management Odoo Apps helps user to give loyalty points to customers based on payments for specific amount of period by sales, User can redeem this loyalty points via products (Gift Products) inside the sales. This Odoo module also allows user to maintain loyalty history, user can generate various types of reports like customer loyalty history report, customer points report, and redeem receipt report.Sales Order records the Loyalty Rewards based on configuration setup and those rewards will be redeemed on Sales.

Features

Loyalty Point Configuration

User can set expiry date with grace period for loyalty points.

Earn Loyalty Points

Customer will get loyalty points on successful payment.

Redeem Loyalty Points

Customer can redeem loyalty points via products (Gift Products).

Reports

User can track loyalty by generating reports i.e loyalty history report, customer points report, and redeem receipt report.
Demo Request Video Tutorial

Loyalty Management Configuration

Configure loyalty point calculation by entering amount for single loyalty point, select closing year date and grace period date under reset loyalty point configuration.

Create Gift Product

Enable "Gift Product" checkbox and enter number of loyalty points for selected product.

Loyalty Point in Customer

After successful payment, customer will get loyalty points as per product.

User can see earned loyalty points in customer form view.

Redeem Loyalty Points

Under Loyalty Management > Redeem Points menu user can redeem earned loyalty points.

In this wizard user can select customer and redeem loyalty points, "Loyalty Points Have" show total available loyalty points for selected customer, also select products, used loyalty points and left loyalty points.

Redeem Receipt Report

User can see generated redeem receipt report.

New transfer will created on redeem loyalty points with redeem loyalty points product.

Generate a History Report

Under Loyalty Management > Reports > History Report menu user can generate history report for loyalty management.

User can apply different filters like start date or end date, status, product, and customers on history report wizard.

History Report

User can see generated history report with selected filters.

Generate a Customer Points Report

Under Loyalty Management > Reports > Customer Points Report menu user can generate customer points report for loyalty management.

User can apply different filters like start date or end date, status, product, and customers on customer points report wizard.
